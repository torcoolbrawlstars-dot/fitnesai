# 📱 Как собрать .ipa для iPhone через GitHub

iOS-приложение **нельзя собрать на Windows** — нужен Mac. Поэтому мы используем
бесплатные macOS-компьютеры GitHub (GitHub Actions): вы загружаете код на GitHub,
он сам собирает `.ipa` в облаке, и вы его скачиваете.

Всё уже подготовлено: iOS-проект (Capacitor) и файл сборки
`.github/workflows/ios.yml` лежат в проекте.

---

## Шаг 1. Создайте репозиторий на GitHub

1. Зайдите на https://github.com/new
2. Название — например `bodyvision-ai`. Можно **Private** (приватный).
3. Ничего не добавляйте (ни README, ни .gitignore) — просто нажмите **Create repository**.

## Шаг 2. Загрузите код (выполните в этой папке)

Скопируйте команды из раздела на странице созданного репозитория
(«…or push an existing repository»), либо выполните эти, подставив свой логин:

```bash
git remote add origin https://github.com/ВАШ_ЛОГИН/bodyvision-ai.git
git push -u origin main
```

> Репозиторий уже инициализирован и все файлы закоммичены — нужно только `push`.

## Шаг 3. Добавьте ключ Gemini как секрет (чтобы ИИ работал)

Файл `.env.local` с ключом **специально не загружается** на GitHub (иначе ключ
украдут боты). Чтобы ИИ-анализ работал в приложении:

1. В репозитории: **Settings → Secrets and variables → Actions → New repository secret**
2. Name: `NEXT_PUBLIC_GEMINI_API_KEY`
3. Secret: ваш ключ (тот, что в файле `.env.local`)
4. **Add secret**

> Без этого шага приложение соберётся, но ИИ будет показывать расчёт без анализа фото.

## Шаг 4. Запустите сборку

1. Откройте вкладку **Actions** в репозитории.
2. Слева выберите **Build iOS IPA** → справа кнопка **Run workflow** → **Run workflow**.
   (Либо сборка стартует сама после каждого `git push`.)
3. Подождите ~5–10 минут, пока пройдёт зелёная галочка.

## Шаг 5. Скачайте .ipa

1. Откройте завершившуюся сборку во вкладке **Actions**.
2. Внизу, в разделе **Artifacts**, скачайте **BodyVisionAI-unsigned-ipa**.
3. Внутри — файл `BodyVisionAI-unsigned.ipa`.

---

## Как установить .ipa на iPhone

Собранный `.ipa` **без подписи** (unsigned) — это нормально для установки без App Store.
Подпишите и установите его любым из способов:

- **Sideloadly** (https://sideloadly.io) — Windows/Mac, подключаете iPhone кабелем,
  вводите свой Apple ID, перетаскиваете `.ipa`. Приложение живёт 7 дней (бесплатный
  Apple ID), потом переустановить.
- **AltStore** (https://altstore.io) — то же самое, с автопродлением подписи.
- **Xcode** (на Mac) — открыть `ios/App` и запустить на своём iPhone.
- Платный аккаунт Apple Developer ($99/год) — подпись на год и установка «по воздуху».

> App Store требует платный Apple Developer-аккаунт и отдельную процедуру ревью —
> это уже отдельный большой шаг, не входит в сборку `.ipa`.

---

## Если сборка упала

- Проверьте вкладку **Actions** → откройте красную сборку → посмотрите, на каком шаге ошибка.
- Чаще всего помогает просто **перезапустить** (Re-run jobs) — иногда GitHub тормозит на загрузке зависимостей.
- Обновление зависимостей/Xcode: раннер `macos-15` в файле `.github/workflows/ios.yml`
  можно поменять на `macos-14`, если что-то несовместимо.
